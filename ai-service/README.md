# ai-service

Папка под локальную модель и инструкцию её запуска.

## Роль на первом этапе
- здесь может лежать модель, веса, скрипт запуска или инструкция по поднятию локального inference;
- отдельный сетевой AI-сервис в MVP не обязателен;
- все вызовы и оркестрация происходят из `backend-football`.

## Ожидаемое использование
1. backend готовит входные данные по новости;
2. backend вызывает локальную модель или stub;
3. backend получает шаблонный текст поста и возвращает его в Mini App.
